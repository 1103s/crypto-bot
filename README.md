# crypto-bot
Apply machine learning to cryptocurrency data

# Requirements
- python 3.9.1
- Anaconda
- pandas==1.4.0
- Install cryptoCMD (`https://github.com/guptarohit/cryptoCMD`) can be installed using `pip install git+git://github.com/guptarohit/cryptoCMD.git`
- scikit-learn==1.0.2
- numpy==1.22.2
- matplotlib==3.5.1
